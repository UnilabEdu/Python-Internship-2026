რეგისტრაცია

მოგესალმებით მინი ობსერვატორიაში

უსასრულო ღამის ცის ქვეშ, შუაღამის ობსერვატორია მიესალმება ყველა ცნობისმოყვარე სულს, რომელიც ვარსკვლავებს უყურებს და ფიქრობს, თუ რა იმალება მის მიღმა. შორეული გალაქტიკებიდან დაწყებული მანათობელი თანავარსკვლავედებით დამთავრებული, ასტრონომია გვახსენებს, თუ რამდენად უზარმაზარი და იდუმალია სამყარო სინამდვილეში. ეს პატარა ობსერვატორია შეიქმნა ვარსკვლავების დამკვირვებლებისთვის და კოსმოსის მოყვარულებისთვის, რომლებიც სილამაზეს მთვარის შუქში, სიჩუმესა და ვარსკვლავებს შორის უსასრულო სიბნელეში პოულობენ.


ვარსკლავების შესახებ

ვარსკვლავები ათასობით წლის განმავლობაში ხელმძღვანელობდნენ მოგზაურებს, შთააგონებდნენ მითებს და ანათებდნენ ღამის ცას.


მთვარის შესახებ

მთვარე მშვიდად აკვირდება დედამიწას და ღამესთან ერთად იცვლის ფორმას.


სამყაროს შესახებ

იმის მიღმა, რასაც ვხედავთ, უსასრულო სამყაროა, სავსე საიდუმლოებებით, რომლებიც აღმოჩენას ელოდება.
